# Vue Frontの技術要素
 - Vue.js
 - Vuetify
 - Vite
 - Vue Router
 - Socket.io
 - - WebSocketとは異なり、Socket.IOを使用すると、接続されているすべてのクライアントにメッセージをブロードキャストできます。
 - - WebSocketは接続が切れると自動的再接続する。
